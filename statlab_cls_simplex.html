Classification/Multinomial and Simplex

Disclaimer: This page is provided only for studying and practicing. The author does not intend to promote or advocate any particular analysis method or software.

Background

For multi-class classification, the sum of probability of individual classes is restricted to one [1]. This page includes discussions related to the space of probability values on a simplex.

Notation

Let \(Y_{i}=c\) be the class (group) of the \(i\) th sample for \(i = 1, \ldots, n\), with class value \(c \in \{c_1,\ldots, c_K\}\), \(P(Y_i = c_k) = p_k\), \(p_k \in [0,1]\) for all \(k\), and \(\sum_k p_k=1\). Let \(N_k = \sum_{i=1}^n I(Y_i = c_k)\), \(N = (N_1, \ldots, N_K)\) with \(N_k \geq 0\) and \(\sum_k N_k = n\). The vector \(N\) is a \(K\) components compositional data point. Let \(\hat{p} = (\hat{p}_1, \ldots, \hat{p}_K) \in [0,1]^K\) with \(\sum_k \hat{p}_k = 1\). The value space of \(\{\hat{p}\}\) is a \((K-1)\)-dimensional standard (normalized) simplex [2].

How-to

To use scipy.stats [3]:

from scipy.stats import spearmanr

y1 = [1, 3, 2]
y2 = [4, 6, 5]

rho, p_value = spearmanr(y1, y2)
print("Spearman's rho:", rho)
